# 谍影钱庄 (Spy-Bank-MVP)

欢迎加入《谍影钱庄》MVP 开发小队！这是一个基于 **微信小程序 + 云开发 (CloudBase)** 的敏捷开发项目。

## 🚀 快速开始
1. **获取代码**：请所有人 Clone 代码，切到 `develop` 分支。
2. **导入项目**：打开微信开发者工具，导入项目根目录。
3. **环境配置**：确保 `miniprogram/app.js` 中配置了正确的云环境 ID。
4. **必读文档**：新成员请优先阅读 `docs/` 目录下的相关规范。
   - 接口问题查 `docs/04_API协议与实时规范_v1.2.md`
   - 代码风格与 AI 辅助查 `.cursorrules`

## 📁 目录结构
- `miniprogram/`: 📱 前端小程序工作区 (组员 A/B)
- `cloudfunctions/`: ☁️ 后端云函数工作区 (组员 C)
- `docs/`: 📚 项目文档与协同规范
- `.cursorrules`: 🤖 AI 辅助编程规则

## ⚖️ 规则 (Git 协同)
- 禁止直接 Push `main` 或 `develop` 分支。
- 每天切 `feature/姓名-功能` 分支开发。
- 写完代码提 Merge Request 并邀请 Lead 进行 Review。

祝我们旗开得胜！
