# 《谍影钱庄》MVP 功能概览说明书 (v1.0)

**项目名称**：谍影钱庄 (Spy Bank)  
**产品版本**：MVP v1.0 (内部演示版)  
**核心目标**：实现 7 人制谍战桌游的资产实时调度与沉浸式交互。

---

## 一、 准入模块：扫码即达 (The Entrance)
*   **专属身份码**：系统提供 7 个独立的专属二维码，对应 7 个核心角色（如黄金大班、码头教父等）。
*   **静默绑定逻辑**：玩家只需微信扫码，无需注册或手动选角。系统自动识别身份并直接进入个人资产看板。
*   **业务价值**：消除技术门槛，确保游戏开场即入戏，1.5 秒内完成所有玩家上线。

## 二、 玩家模块：实时看板 (Player Dashboard)
*   **复古角色画像**：进入后首屏展示 1940s 复古风格的人物画像与绝密档案。
*   **实时余额跳表**：大洋余额采用“机械跳表”动效。每当资产变动时，数字会实时滚动。
*   **0 延迟到账通知**：当银行（DM）分发资金时，玩家手机会实时显示“红章盖下”动效。
*   **绝密交易存根**：提供分页式的流水账单，记录每一笔钱的来源（系统奖励、情报交易、罚金扣除）。
*   **业务价值**：通过极致的视觉反馈，提升玩家在金融谍战中的沉浸感与获得感。

## 三、 银行/DM 模块：中央调度台 (Bank Console)
*   **资产清算工具**：DM 拥有“中央银行”权限，可对 7 名玩家进行任意资产操作。
    *   **一键拨盘**：支持单选或多选玩家，快速分发任务奖励。
    *   **增减扣划**：手动输入金额，进行罚金扣除或贷款发放。
    *   **清算结案**：游戏结束时一键统计全场资产排名。
*   **账户权限控制**：DM 可以实时锁定/解封某个角色的账户（如：剧情中角色被捕）。
*   **业务价值**：将 DM 从繁琐的人工记账中彻底解放，操作效率提升 50% 以上，降低错账率。

## 四、 核心技术亮点 (Technical Highlights)
*   **金融级安全**：所有资产变动均在云端进行“原子化”校验，杜绝任何数据篡改或并发冲突。
*   **原生性能**：基于微信小程序原生框架，确保在各种型号手机上均有极佳的流畅度与兼容性。
*   **实时推送架构**：基于微信云开发实时推送技术，实现“银行发钱，玩家秒到”的 0 延迟体验。
