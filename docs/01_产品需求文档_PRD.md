# 《谍影钱庄》MVP 产品需求文档 (PRD) v1.0

## 一、 游戏规模 (Game Scale)
- **标准开局**: 8 名特工 + 1 名 DM
- **支持范围**: 4 - 12 人灵活配置
- **多端同步**: 
  - DM 端: 具有管理权限的 PC/平板/手机
  - 玩家端: 一人一机 (微信小程序)

## 二、 核心业务逻辑 (Core Loop)
1. **身份认证**: 玩家扫码进入，根据 DM 预置的 12 个角色槽位进行选角。
2. **资产查看**: 实时查看个人金条、密信、银行卡状态。
3. **资金流动**: 
   - 玩家间转账 (Peer-to-Peer)
   - 银行贷款/还款
   - 交易所买入/卖出

## 三、 视觉风格 (Art Vibe)
- **主题**: 复古谍战 (1940s 风格)
- **色彩**: 报纸灰、印章红、做旧金
- **字体**: 系统默认，配合适当的 CSS 滤镜实现报纸质感

## 四、 关键性能指标 (KPI)
- **实时性**: 转账到账延迟 < 500ms
- **并发性**: 支持 12 人同时在 1 分钟内产生 60 笔以上交易不报错
