# 《谍影钱庄》云数据库表结构 v1.2 (草案)

## 一、 角色表 (Roles)
- `_id`: 字符串 (角色ID, 如 role_01)
- `name`: 字符串 (角色名, 如 "黄金大班")
- `avatar`: 字符串 (云存储路径)
- `isTaken`: 布尔值 (是否已被占用)
- `initialAsset`: 数字 (初始资产)

## 二、 玩家/特工表 (Users)
- `_id`: 字符串 (OpenID)
- `roleId`: 字符串 (关联 Roles 表)
- `balance`: 数字 (当前余额/大洋)
- `status`: 字符串 (正常/锁定/被捕)
- `lastLogin`: 日期

## 三、 交易流水表 (Transactions)
- `_id`: 字符串
- `from`: 字符串 (发款方 OpenID)
- `to`: 字符串 (收款方 OpenID)
- `amount`: 数字 (交易金额)
- `type`: 字符串 (转账/贷款/还款/系统奖励)
- `timestamp`: 日期 (服务器时间)
- `note`: 字符串 (交易备注，密电内容)

## 四、 全局配置表 (Config)
- `_id`: "global_config"
- `gameStatus`: 字符串 (等待中/进行中/已结束)
- `isExchangeOpen`: 布尔值 (交易所是否开启)
